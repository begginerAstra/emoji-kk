# Textos prontos para apresentação

Use estes textos para apresentar a ideia para um cliente, adaptar painéis ou montar mensagens no Discord.

---

## 1. Mensagem de apresentação

```txt
Olá! Preparamos uma demonstração visual para mostrar como o seu servidor pode ter uma identidade própria usando emojis personalizados, painéis organizados e textos profissionais.

A ideia não é apenas colocar emojis nas mensagens. A ideia é transformar os emojis em parte da identidade do servidor, usando cada um deles em funções específicas como tickets, staff, loja, avisos, ranking, VIP, recompensas e sistema de moedas.
```

---

## 2. Explicação curta da ideia

```txt
Nosso objetivo é deixar o servidor mais bonito, organizado e profissional, sem usar mensagens genéricas.

Cada emoji tem uma função visual dentro do servidor, ajudando os membros a entenderem melhor cada área e deixando os painéis com uma aparência mais exclusiva.
```
